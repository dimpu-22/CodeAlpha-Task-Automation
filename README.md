# Task Automation with Python Scripts

## Objective
Automate the process of moving all .jpg files from one folder to another.

## Modules Used
- os
- shutil

## How to Run
1. Place .jpg files inside the Source folder.
2. Run:
   python move_images.py
3. The images will be moved to the Destination folder.

## Output
All .jpg files are successfully moved from the Source folder to the Destination folder.
